⚖ Ajuste Manual de Saldo
Sección: Saldos & Transferencias

Permite corregir diferencias puntuales sin crear una transferencia formal. Úsalo cuando hay un gasto o ingreso excepcional fuera de la planilla.

Valor ingresado Efecto
Negativo (ej: −15000) Registra un gasto extra → aumenta el Total Gastado → reduce el saldo.
Positivo (ej: +10000) Registra un ingreso extra → reduce el déficit o aumenta el saldo a favor.
⚠ El ajuste se acumula. Si lo aplicas dos veces, se suma dos veces. Para corregir un ajuste erróneo, ingresa el valor contrario.
💾 Guardado Automático
Momento Descripción
Al salir de cada celda Guardado inmediato.
1,5 seg tras el último cambio Guardado automático por debounce mientras se escribe.
Al minimizar o cambiar pestaña El sistema guarda al perder el foco.
Al cerrar la página Guardado sincrónico justo antes de cerrar.
Cada 20 segundos Respaldo periódico automático.
Badge en toolbar: Naranja = pendiente · Verde = guardado · Rojo = error
🔒 Sistema de Respaldo

El sistema usa dos mecanismos de persistencia complementarios:

Mecanismo Descripción
localStorage (automático) Los datos se guardan en el navegador automáticamente. Disponibles al reabrir el mismo archivo en el mismo equipo.
Respaldo JSON (manual) Usa ⬇ Respaldar Datos para exportar un archivo .json. Para restaurar usa ⬆ Cargar Respaldo.
💡 Recomendación: Respaldar al terminar cada jornada de ingreso. Guardar el archivo .json en una carpeta fija (iCloud, Escritorio, etc.).
❓ Preguntas Frecuentes
¿Puedo usar el sistema en otro computador?
Sí. Usa ⬇ Respaldar Datos para exportar un .json con todos tus datos, luego ábrelo en el nuevo equipo con ⬆ Cargar Respaldo.
¿Qué pasa si cierro el navegador por accidente?
El sistema guarda automáticamente antes de cerrar. Al reabrir el mismo archivo, los datos se restauran desde el navegador.
¿Por qué el conteo de eventos no coincide con las filas?
Solo se cuentan las filas donde la columna "Nombre Evento" tiene texto. Las fechas sin evento no se cuentan.
Ingresé un ajuste manual equivocado, ¿cómo corrijo?
Ingresa el valor contrario. Si pusiste −15000 por error, ingresa +15000. Los ajustes se acumulan.
¿Para qué sirve "Respaldar Datos" y cuándo debo usarlo?
Genera un .json con todos los datos. Úsalo al terminar el día. Si abres una nueva versión del archivo HTML y los datos no cargan, usa ⬆ Cargar Respaldo.
¿Qué diferencia hay entre "Sincronizar Dashboard" y "Respaldar Datos"?
Sincronizar actualiza los totales del Dashboard internamente. Respaldar exporta todos los datos a un archivo externo para protegerlos.
